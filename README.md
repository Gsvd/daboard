# Daboard

## Project setup
```
cd api && npm i
cd app && npm i
```

Create ``api/.env`` file

```
DB_HOST=
DB_USER=
DB_PASSWORD=
DP_PORT=
DB_DATABASE=

OAUTH_ID=
OAUTH_SECRET=
OAUTH_USERNAME=
OAUTH_PASSWORD=
```

### Compiles and hot-reloads for development
```
cd api && npm start
cd app && npm run serve
```

### Compiles and minifies for production
```
cd app && npm run build
```
